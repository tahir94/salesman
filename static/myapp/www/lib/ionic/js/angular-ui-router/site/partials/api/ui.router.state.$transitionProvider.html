  • on(matchObject, callback)

    Registers a function to be injected and invoked when a transition between the matched 'to' and 'from' states starts. This function can be injected with one additional special value: - $transition$: The current transition

    Parameters
    ParamTypeDetails
    matchObjectobject

    An object that specifies which transitions to invoke the callback for.

    • to - {string|function=} - A glob string that matches the 'to' state's name. Or, a function with the signature function(state) {} which should return a boolean to indicate if the state matches.
    • from - {string|function=} - A glob string that matches the 'from' state's name. Or, a function with the signature function(state) {} which should return a boolean to indicate if the state matches.
    callbackfunction

    The function which will be injected and invoked, when a matching transition is started. The function may optionally return a {boolean|Transition|object} value which will affect the current transition:

    • false to abort the current transition
    • {Transition} A Transition object from the $transition$.redirect() factory. If returned, the current transition will be aborted and the returned Transition will supersede it.
    • {object} A map of resolve functions to be added to the current transition. These resolves will be made available for injection to further steps in the transition. The object should have {string}s for keys and {function}s for values, like the resolve object in $stateProvider.state.
